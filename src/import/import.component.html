To import a snippet, you will need to have one of the following:

  1. A URL for a GitHub Gist that was exported by the Playground. For example:
    https://gist.github.com/{{sampleGistId}}
  2. A "View" URL created by the Playground. For example:
    {{playgroundBasePath}}#/view/{{sampleGistId}}
  3. A manually-pasted JSON that has been exported from an Playground snippet. Something like: 
    {
  "meta": {
    "playgroundVersion": 1,
    "name": "Basic snippet"
  },
  "script": [
    "console.log(\"Hello world\");"
  ],
  ...
}

Ready?

Please paste the Snippet URL or JSON into the text area below, and then choose the "Import" button at the bottom of the screen.
