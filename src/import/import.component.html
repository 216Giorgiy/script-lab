To import a snippet, you will need to have one of the following:

  1. A share ID generated by the Playground. For example:
    gist_3dfeba6704bc6b78004869675dff2b45
  2. A GitHub Gist URL created by the Playground. For example:
    https://gist.github.com/3dfeba6704bc6b78004869675dff2b45
  3. A manually-pasted JSON that has been exported from an Playground snippet. Something like: 
    {
    "meta": {
        "playgroundVersion": 1,
        "name": "Basic snippet"
    },
    "script": [
        "console.log(\"Hello world\");"
    ],
    ...
}
